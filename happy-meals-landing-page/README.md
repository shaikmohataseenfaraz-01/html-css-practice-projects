# Happy Meals Landing Page

A food delivery landing page built with HTML, CSS, and Bootstrap.

## Description
A full-viewport landing page featuring a food background image, a card container with the title, description, and a Book Now button.

## Technologies Used
- HTML
- CSS
- Bootstrap 4
- Google Fonts (Bree Serif, Roboto)

## Preview
Open `index.html` in your browser to view the project.
