# Royal Grand Hotel

A hotel booking landing page built with HTML, CSS, and Bootstrap.

## Description
A full-viewport landing page featuring a luxury hotel background image, hotel name, location (R.K Beach, Visakhapatnam), price, and a Book Now button.

## Technologies Used
- HTML
- CSS
- Bootstrap 4
- Google Fonts (Bree Serif, Roboto)

## Preview
Open `index.html` in your browser to view the project.
