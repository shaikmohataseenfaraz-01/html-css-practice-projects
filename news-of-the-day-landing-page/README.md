# News of the Day Landing Page

A news landing page built with HTML, CSS, and Bootstrap.

## Description
A full-viewport landing page featuring a news background image, an article snippet about the iB Cricket Virtual Reality Cricket League, and a Read More button.

## Technologies Used
- HTML
- CSS
- Bootstrap 4
- Google Fonts (Roboto)

## Preview
Open `index.html` in your browser to view the project.
