# Diwali Celebration Shop

A Diwali shopping landing page built with HTML, CSS, and Bootstrap.

## Description
A festive page featuring a Diwali fireworks background with 4 product cards — Diya Lamp, Diwali Crackers, Fireworks, and Diwali Decoration — along with View More and Buy Now buttons.

## Technologies Used
- HTML
- CSS
- Bootstrap 4
- Google Fonts (Caveat, Roboto)

## Preview
Open `index.html` in your browser to view the project.
